電力需給のお知らせ

関西エリアの需給実績の公表について

経済産業省　資源エネルギー庁の「系統情報の公表考え方」に基づき、関西エリアの需給実績を公表します。

CSVダウンロード

エリア需給実績
  • 2016年度 [CSVファイル 505,212B]
  • 2017年度 [CSVファイル 527,117B]
  • 2018年度 [CSVファイル 531,086B]
  • 2019年度(4月～10月) [CSVファイル 314,458B]

取り扱い

  • 需給実績は毎月公表します。
  • 例えば、実績9:00とは、9時～10時までの1時間平均の電力を表示したものです。
  • 発電実績は速報値であり、推計実績を含んでいます。
  • 公表値は、過去にさかのぼり修正させていただく場合があります。
  • 過去の公表値を修正する場合、実績修正のお知らせや修正前の実績の提供はいたしませんので、予め了承ください。
  • CSVデータの仕様については、予告なく変更することがあります。
  • 四捨五入の関係で合計が合わない場合があります。
  • 「火力」は、LNG/LPGおよび石炭、石油、混燃バイオマス他、電源種別が不明なものを含みます。
  • 「水力」は、揚水発電は含みません。
  • 「揚水」は、発電量(+計上)と揚水動力量(-計上)を合算したものとなります。
  • 「太陽光」、「エリア需要」は、太陽光発電の自家消費分は含みません（自家消費分は需要減少として表れている）。

免責事項

  • 当システムは、メンテナンス等で予告なく停止する場合があります。
  • 複数のミラーサーバを利用しているため、タイミングによって、古い情報が表示されることがあります。
  • 当社は、本サービスの利用者が、本サービスに依拠し、または本サービスの情報を信頼して行った行動等により被った、いかなる生命、身体、財産上の損失又は損害についても責任を負いかねますので、予めご了承ください。
